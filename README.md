# Interactive Infographics — Portfolio

Five single-page interactive infographics, each built as a self-contained HTML document. No frameworks, no build step, no JS dependencies beyond Google Fonts.

## Pieces

| # | Folder | Style | Subject |
|---|--------|-------|---------|
| 01 | `01-solar-system-explorer/` | Atlas | Animated tour of the eight planets |
| 02 | `02-ocean-depths/` | Cutaway / Scroll | Vertical descent from surface to Challenger Deep |
| 03 | `03-programming-languages/` | Network / Timeline | 30 languages and their lines of influence |
| 04 | `04-smartphone-anatomy/` | Anatomy | Interactive cutaway of a modern smartphone |
| 05 | `05-on-the-same-scale/` | Comparison | Things drawn at true relative scale |

## Run locally

Each piece is a single HTML file. Just open the `index.html` inside any folder in a browser:

```
open 01-solar-system-explorer/index.html
```

No server needed.

## Live demo

Once GitHub Pages is enabled on this repo, all five are accessible from the root `index.html` landing page.
