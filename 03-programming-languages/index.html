Popularity over time — nothing stays on top forever.

A Map of Strictness

    
Where each language sits on the type spectrum.

    
Two axes that explain a lot of fights between programmers. Static vs Dynamic: when does the language check types — at compile time, or at runtime? Strong vs Weak: how willing is the language to silently convert one type into another? Hover over any language to read more.

One Problem, Six Translations

    
The same recursive Fibonacci, in six languages.

    
A canonical exercise: compute the n-th Fibonacci number recursively. Functionally identical, syntactically wildly different. Notice how Python and Haskell collapse to almost a single line, while Java and C++ require ceremony before any logic appears.
